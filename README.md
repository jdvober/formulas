# Formula Sheet

## Formula Formats

### Format A

$a=\frac{b}{c}$

### Format B

$a=b \cdot c$

### Format C

$a \cdot b=c \cdot d$

### Format D

$a = \frac{b \cdot c}{d}$

### Format E

$a = b \cdot c \cdot d$

### Format F

$a = \frac{b}{c \cdot d}$

### Format G

$a = \sqrt{\frac{b \cdot c}{d}}$
